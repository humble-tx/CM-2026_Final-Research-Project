# Technical Description

The project was developed using Unity and C#. Communication between hardware and software was implemented through Python serial scripts and Bluetooth transmission.

For physical interaction, I experimented with Arduino, MPU6050 IMU sensors, electromagnets, MOSFET circuits and battery-powered prototypes. The IMU was used to detect rotational movement, while the electromagnet provided impact feedback to simulate collisions.

Early prototypes used the vibration motors of a PS5 controller before moving toward a custom spherical controller. User testing and iterative prototyping guided the development throughout the project.
